# Governance — Open Source Stewardship

> 이 저장소는 개인 학습 노트로 시작했으나, **장기 지속성** 확보를 위해 오픈소스 거버넌스 모델로 전환 중. 마지막 갱신: 2026-04-23.

## 🎯 Mission

**"한국 가상자산 산업의 AML 학습·실무 표준 자료를 공개·무료·다국어로 제공"**

---

## 👥 Roles

### Maintainer (현재 1명: lala-david)
- 콘텐츠 방향성·품질 책임
- PR 최종 승인
- 분기 리뷰 진행

### Contributor (모집 중)
- PR·Issue 통한 기여
- 5+ accepted PRs → "Trusted Contributor"
- Trusted → 특정 폴더 review 권한

### Reviewer (분야별, 모집 중)
- **법무 Reviewer**: 변호사·법학자 (법 조문 정확도)
- **기술 Reviewer**: AML 엔지니어 (pseudocode·아키텍처)
- **실무 Reviewer**: AMLO·분석가 (현장 적용성)
- **번역 Reviewer**: 영어·일본어·중국어 native

### Sponsor (선택)
- GitHub Sponsors / Open Collective
- 기금은 (1) 분기 리뷰 시간 (2) 외부 검토 fee (3) 인프라 비용에 사용
- 모든 sponsor 공개 명시 (개인·회사)

---

## 📋 Decision Process

### 일상 결정 (typo·링크 수정·minor 추가)
- Maintainer 단독 결정
- Contributor PR 1주일 내 응답

### Major 결정 (구조 변경·신규 카테고리·라이선스 변경)
- GitHub Discussion으로 1주일 공개 토론
- Reviewer 의견 수렴
- Maintainer 최종 결정

### Critical 결정 (저장소 이전·거버넌스 변경)
- 30일 공개 토론
- 모든 Trusted Contributor 투표 (만약 5명 이상 시)

## 🚪 Conflict Resolution

- **콘텐츠 정확성 분쟁**: Reviewer 다수결 (분야별)
- **방향성 분쟁**: Discussion 공개 토론 + Maintainer 결정
- **Code of Conduct 위반**: Maintainer 즉시 차단
