## Tips

- **처음 포스트는 겸손하게** — "배포" 톤, "완성" 톤 금지
- **명확한 CTA** — 한 번에 하나만 요청 (Star·Issue·PR·Feedback 중 1개)
- **개인 브랜드 연결** — 본인 소속·경력 자연스럽게 언급
- **응답 의무** — 댓글·DM 24시간 내 회신
- **스팸 금지** — 한 채널 동일 포스트 재발행 금지, 업데이트만
- **타임존 최적화** — 한국 화~목 오전 9시 / 미국 화~목 오전 10시 EST

## 응답 추적

각 포스트 URL과 주요 지표 기록 (별도 `00-metrics-log.md` 권장):

- Views / Impressions
- Engagements (좋아요·공유·댓글)
- 신규 GitHub Star / Fork
- 신규 Issue / PR
- 신규 Contributor (5+ PR 기준 trusted)
