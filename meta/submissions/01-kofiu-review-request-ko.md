본 자료는 22,000여 줄 규모의 한국어 AML 학습 교재로서, 특히 다음 분야를 다룹니다.

3. **감독 검사 4주 대응 워크북** (`notes/5-compliance/inspection-response.md`, 1,200여 줄)
4. **STR·EDD·RBA 실무 템플릿**

본 자료는 **Creative Commons BY 4.0 라이선스**로 무료 공개하며, 한국 신입 AMLO·
실무자·학생의 표준 학습 경로 구축을 지향합니다.

**정확성을 확보**하기 위해, KoFIU의 **공식 검수 의견**을 부탁드리고자 합니다.

#### 검수 요청 사항
1. **조항 해석 정확성** — 특히 특금법 §5-2·§5-3·§7·§15 관련 해석
2. **STR 양식 및 품질 기준 부합 여부** — FIU-TIS 표준 대비
3. **검사 대응 워크북의 실무 부합성** — KoFIU 실제 검사 프로세스 대비
4. **2026-01 시행 대주주 심사 강화 반영 정확성**

#### 제안 방식 (중 택 1)
- **(A) 10분 회의** — 화상 또는 대면, 주요 이슈 3~5개 간단 검토
- **(B) 서면 회신** — 핵심 문서 1~2개 검토 의견만 회신
- **(C) 지정 담당자 연결** — 분야별 정책관 소개

#### 후속 조치 약속
- 검수 의견 발견 사항은 **7일 내 GitHub 반영**
- "KoFIU 정책기획팀 자문: YYYY-MM-DD" 표기 (동의 시)
- 본 자료가 KoFIU 자료 배포에 참고될 수 있다면, 추가 공동 작업도 환영
