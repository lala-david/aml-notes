- 60일 커리큘럼의 **D54 (논문) + D55 (리포트)** 가 이 폴더 활용
- 그 이후엔 **자유 학습용 깊이 자료**
- 새 논문/리포트 발견 시 여기 추가

## 학습 방법

학술 논문 읽기 (1편 ~ 60~90분):
1. **추상 (5분)** — 무엇을 주장하는가
2. **결론 (10분)** — 무엇을 발견했는가
3. **방법 (20분)** — 어떻게 했는가 (수식 패스 OK)
4. **관련연구 (10분)** — 학계 위치
5. **메모 (15분)** — 한 줄 요약 + 한계 + 산업 연관성
