## 학습 방법 — 논문 5단계 속독법

학술 논문은 처음부터 끝까지 읽지 않아도 됩니다. 다음 5단계 순서가 효율적입니다 (총 60~90분):

1. **추상 (5분)** — 무엇을 주장하는가
2. **결론 (10분)** — 무엇을 발견했는가
3. **방법 (20분)** — 어떻게 했는가 (수식 패스 OK)
4. **관련연구 (10분)** — 학계 위치
5. **메모 (15분)** — 한 줄 요약 + 한계 + 산업 연관성

### 왜 이 순서인가

논문의 본문(Method·Experiments)은 학자끼리 통하는 언어로 쓰이므로 처음 읽으면 이해가 어렵습니다. 먼저 **추상·결론**으로 "이 논문이 왜 쓰였고 무엇을 증명했는지" 뼈대를 잡은 뒤, 방법을 보면 훨씬 친숙해집니다. 이건 학술 독서의 보편 기술이며, 박사 과정에서도 쓰는 방법.
