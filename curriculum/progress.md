# Progress Tracker

> **60일 챌린지 진척도**. 매일 체크 + 한 줄 메모. 이 트래커는 단순 완료 체크가 아니라 **"무엇을 배웠고 어디가 약한가"** 를 스스로 진단하는 도구입니다.

## 사용법

1. **시작일·목표 종료일**을 먼저 적어두세요 — 완주 의지의 심리적 기준선.
2. 매 day 시작 시 **🗺 오늘의 지도** Mermaid 다이어그램을 30초 응시 → 구조를 먼저 머리에 얹고 본문 진입.
3. 매 day 종료 시:
   - 체크박스 `[ ]` → `[x]`
   - **한 줄 메모**: 그날 가장 인상 깊은 사실 1개 (숫자·사례·조문 등)
4. 매주 끝(D7·14·21·...)에 **주간 체크포인트** 작성.
5. D30·D60에는 **월말 회고**.
6. 미니 프로젝트 완료 시 `projects/0X-*/` 폴더 생성·git 커밋.

### 완료 기준 정의

- **day 완료** = 🗺 지도 응시 + 체크포인트 모두 체크 + 한 줄 메모 작성 + (해당 시) 미니 챌린지 산출물 저장
- **week 완료** = 7개 day 모두 완료 + 주간 회고 작성
- **미니 프로젝트 완료** = 코드 작동 + sample output 저장 + git commit
