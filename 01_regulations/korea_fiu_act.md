# 한국 — 특정금융정보법 (특금법)

> 한국 가상자산 AML의 헌법. VASP 신고제, KYC, Travel Rule의 법적 근거. 마지막 업데이트: 2026-04-17.

## TL;DR
- **정식명칭**: 「특정 금융거래정보의 보고 및 이용 등에 관한 법률」 (특금법, FIU법)
- **2021-03-25 개정**으로 가상자산사업자(VASP) 신고제 도입 → **2021-09-25 시행**
- VASP 의무: **신고, KYC, AML/CFT, Travel Rule, STR/CTR, 기록보관**
- **2026-01-29 개정안 국회 통과**: VASP 진입규제 강화, **대주주까지 자격심사 확대**
- 감독기구: **금융정보분석원(FIU)** — 금융위원회 산하
- 위반 시: **5년 이하 징역 또는 5천만원 이하 벌금** (미신고 영업)

---

## 1. 법의 정체성

| 항목 | 내용 |
|---|---|
| 약칭 | 특금법, FIU법 |
| 정식명 | 특정 금융거래정보의 보고 및 이용 등에 관한 법률 |
| 제정 | 2001-09-27 (테러자금조달방지 + AML 통합법) |
| 2020-03-24 개정 | **가상자산사업자(VASP) 정의 신설** |
| 2021-03-25 개정 시행 | **VASP 신고제 본격 도입** |
| 2026-01-29 | 진입규제 강화 개정안 국회 통과 |
| 소관 부처 | 금융위원회 |
| 집행 기관 | **금융정보분석원(FIU, Korea Financial Intelligence Unit)** |

→ 원래는 일반 금융기관 대상이었는데, 가상자산을 포섭하기 위해 개정한 게 현재 형태.

## 2. VASP의 정의 (특금법 §2.1.하)

가상자산과 관련하여 다음 행위를 **영업으로** 하는 자:

1. **매도·매수**
2. **다른 가상자산과의 교환**
3. **이전** (외부 지갑/타 사업자로의 이동)
4. **보관·관리** (수탁, custody)
5. **위 1~4 행위의 중개·알선·대행**
6. 그 밖에 **시행령으로 정하는** 행위

→ 수탁업자(custody), 거래소(매수도+교환+이전), OTC desk(중개)가 모두 포함.

**제외:**
- 단순 발행자
- 자기계산 거래만 하는 자 (VASP 정의 미충족)
- DApp/DEX 자체 (운영자 식별 가능 시 포함 검토)

## 3. VASP 신고 의무 (§7)

### 신고 요건
- **금융정보분석원장(FIU)에 신고서 제출**
- 신고 수리 → 영업 가능. 미신고 영업은 **형사처벌**.

### 신고 시 갖춰야 할 것
- **정보보호관리체계(ISMS) 인증** (한국인터넷진흥원 KISA 발급)
- **실명확인 입출금계정** (= 은행 실명계좌, 거래소만 해당)
- **AML/CFT 내부통제기준** 수립
- **자금세탁방지 보고책임자(임원급)** 임명
- 대표/임원 결격사유 부재

### 2026-01-29 개정 핵심: 대주주까지 자격심사
- 종전: 대표·임원만 결격사유 심사
- 개정: **대주주까지 확대** — 사회적 신용/재무상태/내부통제 심사
- 부적격 시 신고 거부/취소 가능

### 신고 갱신
- **3년마다 갱신** (시행령에 따라)

## 4. AML/CFT 의무 (§5의2~§5의4)

### 고객확인(KYC)
- 신원확인: 실명, 생년월일, 주소, 연락처
- 실소유자 확인: **법인은 25% 이상 지분 자연인**
- 거래목적/자금원천 확인
- **재이행 의무**: 위험등급 변동 시, 일정 주기마다

### 강화된 고객확인 (EDD)
- 트리거: PEP, 고위험국 거주, 비대면 + 고위험, 거액거래
- 추가: 자금원천 증빙, 고위경영진 승인

### 거래 모니터링
- 비정상 거래 패턴 탐지 의무
- 자체 모니터링 시스템 구축 또는 외부 위탁

### 의심거래보고 (STR, §4)
- 자금세탁 의심 거래는 **지체 없이** FIU에 보고
- 금액 무관

### 고액현금거래보고 (CTR, §4의2)
- **1천만원 이상 현금거래**는 자동 보고
- 가상자산은 적용 모호 (현금 X)

### 기록보관
- 거래기록: 5년 (특금법 기준)
- **단, 가상자산이용자보호법 §11: 가상자산 거래기록은 15년**
- 더 긴 쪽이 우선

## 5. Travel Rule (§5의2 + 시행령 §10의10)

### 핵심
- VASP가 다른 VASP에게 가상자산을 이전할 때
- **이전 금액이 100만원 이상** 이면 송수신인 정보 제공 의무

### 제공해야 할 정보
- 송신인: 성명, 가상자산주소
- 수신인: 성명, 가상자산주소

### 시행
- **2022-03-25 시행**
- 한국 4대 거래소(업비트/빗썸/코빗/코인원)는 **VerifyVASP** + **CODE** 두 솔루션으로 연동

### 미연결 VASP / 외부 wallet
- 연결 안 된 외국 VASP에게 보내는 경우: 정보 제공 불가 → **송금 거절** 또는 **자체 검토 후 보류**
- **개인지갑(unhosted wallet)**: 정보 수집 어려움 → 외부지갑 등록제 운영 (업비트 등 거래소가 자체 시행)

## 6. 감독·처벌

### 감독
- **FIU**: 검사·자료 제출 요구
- **금융감독원**: 위탁 검사 가능

### 행정처분
- 시정명령
- 영업 일부 또는 전부 정지
- 신고 취소

### 형사처벌 (§17)
- **미신고 영업**: **5년 이하 징역 또는 5천만원 이하 벌금**
- KYC 위반, STR 미보고: 1년 이하 징역 또는 1천만원 이하 과태료
- 비밀누설: 더 무거운 처벌

### 양벌규정
- 법인도 처벌, 임직원도 처벌

## 7. 시행령 / 규정 / 가이드라인

법 위에 다음이 깔려 있음 (실무는 사실 이걸 봐야 함):

1. **특금법 시행령** (대통령령)
2. **특정 금융거래정보 보고 및 감독규정** (FIU 고시)
3. **가상자산사업자 신고 매뉴얼** (FIU 발간)
4. **자금세탁방지 업무규정** (FIU 고시) — 일반 AML 업무규정
5. **위험기반접근법(RBA) 처리기준 가이드라인**

→ 회사에서 신고 준비할 때 **신고 매뉴얼**이 사실상 바이블.

## 8. 미신고 사업자 단속 (2025-12-02 FIU 보도자료)

- 2025년 말 FIU가 **불법 가상자산 취급업자 단속** 강화 발표
- 신고 없이 운영하는 외국 VASP의 한국 고객 영업 차단
- 한국 거래소에 미신고 VASP 주소 차단 협조 요청

## 9. 2단계 입법 — 향후 방향

가상자산이용자보호법 제정 시 부대의견으로 다음이 추진 중:

- 가상자산 발행·유통 분리 (이해상충 해소)
- **스테이블코인 규율** (별도 법 또는 통합법)
- 가상자산 평가업/자문업/공시업 규제
- 시장 인프라 (보관·청산·결제) 정비

→ 수탁업자에게는 **자기자산-고객자산 분리, 운영 기준** 등이 추가 명시될 가능성.

## 10. 회사 컴플라이언스 체크리스트

```
□ FIU 신고 완료 + 3년 주기 갱신 일정 관리
□ ISMS 인증 유지
□ 자금세탁방지 보고책임자 임명 (임원)
□ 내부통제기준 + AML 정책 문서화
□ KYC/CDD/EDD 절차 운영
□ 거래 모니터링 시스템 (자체/외부)
□ Travel Rule 솔루션 연동 (VerifyVASP/CODE)
□ STR/CTR 보고 프로세스
□ 제재 스크리닝 (OFAC/UN/EU + 외교부)
□ 임직원 AML 교육 (연 1회 이상)
□ 거래기록 15년 보존 (이용자보호법 §11)
□ 2026-01 개정 대주주 자격심사 대비 (주주 변동 시)
```
